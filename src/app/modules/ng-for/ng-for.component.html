NgFor works!

*ngFor example

  • This is a built in "Structural Directive".

  • This is used to modify the DOM based on an expression evaluation.

  • This is similar to the for loops used in other programming languages.

  • Allows us to loop over an array/object and display them.

  • The below example loops over the "users" array and displays them on the view.

  • Eg: -

    User Id User Name Index First Last Even Odd
    {{ user.userId }} {{ user.userName }} {{ i }} {{ f }} {{ l }} {{ e }} {{ o }}

*ngFor With local variables

  • "index as i" gives us the index values of the iterable.

  • "first as f" gives us true value where the index is the first in the iterable.

  • "last as l" gives us true value where the index is the last in the iterable.

  • "even as e" gives us true value where the index is the even in the iterable.

  • "odd as o" gives us true value where the index is the odd in the iterable.
