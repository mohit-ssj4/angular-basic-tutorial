NgSwitch works!

*ngSwitch example

  • This is a built in "Structural Directive".

  • This is used to modify the DOM based on an expression evaluation.

  • This is similar to the switch statements used in other programming languages.

  • Allows us to add or remove content dynamically.

  • Unlike if statement, ngSwitch can take in multiple value params for conditional check.

  • There are mainly three components of ngSwitch: -

    • ngSwitch

    • ngSwitchCase

    • ngSwitchDefault

  • The cases which match with the expressions will get executed.

  • If no cases match the expression then the, default case will be executed.

  • Eg: -

    Case 1 shows up if "conditionExpression" is "case1".

    Case 2 shows up if "conditionExpression" is "case2".

    This is the default case if no "conditionExpression" matches with the "ngSwitchCase".
