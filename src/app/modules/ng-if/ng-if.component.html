NgIf works!

*ngIf example

  • This is a built in "Structural Directive".

  • This is used to modify the DOM based on an expression evaluation.

  • If the expression evaluates to true the content will show, else it will not show.

  • Allows us to add or remove content dynamically.

  • Eg: -

    This shows up if the "showContent" expression evaluates to true.

*ngIf With else example

  • Used along with *ngIf.

  • Else statement shows up if the evaluation of the expression is false.

  • In order to use this we need to use "ng-template" and pass the template reference variable along with it.

  • Eg: -

    This shows if the expression evaluates to true;

    This shows if the "showContent" expression evaluates to false.

*ngIf With then and else example

  • Used along with *ngIf.

  • In order to use this we need to use "ng-template" and pass the template reference variable along with it.

  • When the expression evaluates to true, using template reference variable the "then" block will be executed.

  • When the expression evaluates to false, using template reference variable the "else" block will be executed.

  • Eg: -

    This is the then block. This shows up if the "showContent" expression evaluates to true.

    This shows if the "showContent" expression evaluates to false.
