NgClass works!

ngClass example

  • This is a built in "Attribute Directive".

  • This is used to add classes to an element.

  • We can also pass dynamic values via a variable.

  • We can pass multiple values as well.

  • Eg: -

    This element has "text-purple-600" class applied to it.

    This element has "bg-purple-600, text-white" class applied to it.

    This element has "text-red-600" class applied to it based on the expression evaluation.
