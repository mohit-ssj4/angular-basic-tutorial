Interpolation works!

Interpolation example

  • This is a way to bind data between the component and the view.

  • Allows us to communicate with data between component and the view.

  • This is a type of "One-Way Data Binding".

  • The data flows from the "Component to the View".

  • Any type of data can be bound to the view.

  • Using "Interpolation" some simple expression evaluation can also be done.

  • In the below example "message" variable is being bound and displayed in the view.

  • Eg: -

    {{ message }}
