Pipes works!

Pipes

  • Pipes are used to transform the data.

  • They take an input and transform it into our desired format.

  • They are written using the Pipe (|) operator.

  • There are many in-built pipes available in Angular, but we can make custom pipes of our own.

  • We can pass parameters to the pipes as well.

  • We can also chain the pipes to perform multiple operations.

Pipes example

  • These pipes are provided to us by Angular.

  • Some of these are: - uppercase, lowercase, date etc.

  • Eg: -

    {{ username | uppercase }} - In this example we transform the "username" variable to uppercase using "uppercase" pipe.

    {{ dateValue | date }} - In this example we show the today's date in a specific format.

    {{ currencyValue | currency }} - In this example we show the currency value along with the type of currency symbol. By, default the curreny is set to "$".

    {{ currencyValue | currency: "INR" }} - In this example by, passing a parameter like:- " value | pipe_name: "parameter " we modify the output.

    {{ dateValue | date: "short" }} - In this example we modify the date and show in the short form using "short" as a parameter.

    {{ dateValue | date | uppercase }} - This is an example for chaining multiple pipes. First "date" pipe executes and then the "uppercase" pipe will execute.

    {{ username | uppercase | shorten }} - This is an example for custom pipe. Here, the "shorten" pipe is a custom pipe which will display the string in short form if it exceeds 5 characters.
