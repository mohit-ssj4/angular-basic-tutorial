LazyLoading works!

LazyLoading

  • By default the modules are eagerly loaded. Which means as soon as the application is loaded all the modules get loaded.

  • Loading all the modules eagerly, will make the application slow in performance.

  • Lazyloading loads only the modules which are absolutely necessary, and rest we will load based on users interaction.

  • It helps us to decrease the application build size which in turn decreases the load times.

  • Syntax: - ng g module modules/lazyLoading --module app --route lazyLoading
