Databinding works!

DataBinding

  • This is a way to bind data between the component and the view.

  • Allows us to communicate with data between component and the view.

  • There are two types of DataBinding: -

    • One-Way DataBinding (Data flows only in one way)

      • Component To View

        • Interpolation

        • Property Binding

        • Style Binding

        • Attribute Binding

      • View To Component

        • Event Binding

    • Two-Way DataBinding (Data flows in both ways)

      • ngModel Binding ([(ngModel)])
