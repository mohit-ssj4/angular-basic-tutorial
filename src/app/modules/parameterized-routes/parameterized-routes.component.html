ParameterizedRoutes works!

ParameterizedRoutes

  • We can configure the routes and send parameters with them.

  • Also, called as "Dynamic Routes".

  • We can send any type of data in the url.

  • We need to configure the dyamic parameters in the routes as follows: -

    path: "product/:id"

  • Eg: -

    /product/10

  • We can then read these parameters in the component class and process them by subscribing the params.

    This "id: {{ id }}" is coming from the URL param.
