AttributeBinding works!

AttributeBinding example

  • This is a way to bind data between the component and the view.

  • Allows us to communicate with data between component and the view.

  • This is a type of "One-Way Data Binding".

  • The data flows from the "Component to the View".

  • Any HTML attribute can be used for "AttributeBinding".

  • In order to use AttributeBinding we need to use "[attr.arrtibute_name]" brackets on the property.

  • This can be used to create custom attributes/properties for an HTML element.
