NgModelBinding works!

NgModelBinding example

  • This is a way to bind data between the component and the view.

  • Allows us to communicate with data between component and the view.

  • This is a type of "Two-Way Data Binding".

  • The data flows from the "View to the Component" and "Component to the View" as well.

  • Usage: - [(ngModel)] = ""

  • "ngModel" is not available to us by default. So, we need to import the "FormsModule" in that particular module.

  • It is a combination of "Property" and "Event" binding.

  • In the below example the variable "username" has "Two-Way Binding" applied to it. So, any changes you make on the view will be reflected in the component as well and vice-versa.

  • Eg: -

    Username: {{ username }}
