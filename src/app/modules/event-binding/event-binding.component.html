EventBinding works!

EventBinding example

  • This is a way to bind data between the component and the view.

  • Allows us to communicate with data between component and the view.

  • This is a type of "One-Way Data Binding".

  • The data flows from the "View to the Component".

  • Can be used to bind all available events like:- onClick, onBlur etc.

  • Usage: - (click)="functionName()"

  • In the below example on the click of the buttons an "onClick" event is bound to a method in the component.
