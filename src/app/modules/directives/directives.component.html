Directives

  • Directives are mainly of two types: -

    • Structural Directives

    • Attribute Directives

Structural Directives

  • They are used to change the structure of the DOM.

  • They always starts with "*" symbol.

  • Eg: -

    *ngFor(for loop), *ngIf(if-else), *ngSwitchCase(switch case) etc.

Attribute Directives

  • They change the attribute of the element they are applied to.

  • Eg: -

    ngClass, ngStyle etc.

*ngIf example

  • This allows us to modify the structure of the DOM.

  • Eg: -

    This shows up only if the 'showContent' property is true.

ngStyle example

  • This allows us to change the style of the element they are used on.

  • Eg: -

    This content is styled using the ngStyle attribute directive.

ngClass example

  • This allows us to add dynamic classes based on an expression evaluation.

  • Eg: -

    This p tag contains a class 'text-purple-600' based on an expression evaluation.
