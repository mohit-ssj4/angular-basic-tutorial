RouterLink works!

RouterLink

  • This is a built-in "Attribute Directive".

  • This is used to navigate to the configured routes in angular applications.

  • Syntax: - [routerLink]="['/user/bob']"

  • Eg: -

    Static Link: - ParameterizedRoutes

    Dynamic Link: - ParameterizedRoutes

  • We can pass multiple parameters as well.

Redirect Routes Example

  • If we want a route to re-direct to some other we will implement "redirectTo" in our routes array.

  • Syntax: - {{ '{ path: "", redirectTo: "directives", pathMatch: "full" }' }}
